# UX Review role brief

Translate the PM brief into a truthful persona experience bar. Review real App Router routes/components, not mockups alone. Require one obvious next action, owner, waiting party, visibility, prepared output, review boundary, proof destination, next state, failure recovery, and no internal narration on persona surfaces.

Review desktop 1440, mobile 390, and mobile 360 together. Check keyboard/focus, announcements, target size, wrapping, overflow, hydration/console behavior, Cormorant Garamond/Montserrat typography, warm ivory surfaces, dusty state colors, privacy, and persona scope. Return PASS, PARTIAL, or FAIL with an exact Engineering handoff.
